Esempio Postman: download

Aggiorna dati

Endpoint test: test.services.ltc-logistics.it/logica/rest/utente/update
Endpoint produzione: ws.services.ltc-logistics.it/logica/rest/utente/update
Tipo: PUT
Descrizione: Questo end point serve ad aggiornare le proprie informazioni presenti a sistema.
È possibile aggiornare la propria password, il nome, il cognome e l'indirizzo email
Esempio Postman: download

Campi previsti

Nome Richiesto Tipo Descrizione
utenza necessario complesso Va inserito l'oggetto JSON che rappresenta le informazioni sulla propria utenza da aggiornare. Non inserendo un campo viene lasciato il valore già esistente. Se si desidera cambiare password va passato il nuovo valore sul campo nuovaPassword.

Utenza

Campi previsti

Nome Richiesto Tipo Descrizione
username N/A testo Username identificativo dell'utente.
nome N/A testo Il nome dell'utente.
cognome N/A testo Il cognome dell'utente.
email N/A testo L'indirizzo email dell'utente. Eventuali comunicazioni saranno spedite a questo indirizzo.
nuovaPassword N/A testo La nuova password che rimpiazzerà quella vecchia in fase di aggiornamento.
sedi N/A array di numerici interi La lista delle sedi in cui l'utente è abilitato.
commesse N/A array di numerici interi La lista delle commesse in cui l'utente è abilitato.
permessi N/A array di numerici interi La lista dei permessi dell'utente.
features N/A array di stringhe La lista delle features per cui l'utente è abilitato.
